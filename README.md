# SKY Touch Food Website

This is a responsive restaurant website built using **HTML and CSS**.
The project presents a food business with multiple pages such as Home, About, Menu, and Contact, including navigation, styled sections, and an embedded Google Map for location.

---

## Getting Started

### Installation

Clone the repository

git clone https://github.com/yourusername/sky-touch-food.git

Open the project folder and run the website by opening **index.html** in your browser.

---

## Project Structure

* `index.html` – Home page
* `about.html` – About the restaurant
* `menu.html` – Food menu section
* `contact.html` – Contact information and message form
* `styles/` – All CSS files for layout and styling
* `images/` – Images used in the website

---

## Features

* Multi-page website (Home, About, Menu, Contact)
* Responsive navigation bar
* Restaurant hero section and features section
* Food menu display with images and descriptions
* Contact page with form layout
* Google Maps location integration
* Footer with social media icons
* Reusable CSS components and helper classes

---

## Tech Stack

* HTML5
* CSS3
* Font Awesome (icons)
* Google Fonts
* Google Maps Embed
